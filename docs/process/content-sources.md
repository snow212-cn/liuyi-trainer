# 内容来源记录

这个文档用于固定记录仓库内训练正文、插图、音频或其他非代码素材的来源信息。

## 使用规则

- 每次新增、替换或批量更新素材时，都补一条记录
- 记录至少包含来源名称、获取方式、用途范围和维护备注
- 如果某类内容是构建 APK 的关键输入，不要只在聊天或临时笔记里说明
