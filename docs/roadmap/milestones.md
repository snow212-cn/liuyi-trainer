# 里程碑计划

## M0 项目启动

目标：

- 明确产品边界
- 建仓
- 建立架构基线
- 建立内容模型

退出标准：

- 文档齐备
- Android 工程骨架可继续扩展

## M1 训练引擎原型

目标：

- 完成本次训练状态机
- 完成单动作多组记录
- 完成 `2-1-2` 实时节奏驱动计次
- 完成实时阶段显示与语音提示原型
- 完成组间休息倒计时与超时正计时原型
- 完成轻量原型与 Android 逻辑对齐

退出标准：

- 可在本地模拟一批训练并得出每组与总计结果

## M2 MVP

目标：

- 动作选择页
- 训练执行页
- 本次训练总结页
- Room 持久化
- 历史记录列表与详情
- 远程构建首个 APK

退出标准：

- 用户可完整走通一次训练并在历史中回看

## M3 内容系统

目标：

- 动作标准页
- 示意图资源接入
- 内容状态管理

退出标准：

- 六艺十式都可打开对应标准页

## M4 发布准备

目标：

- 真机体验优化
- 音频/振动提示
- 前后台恢复
- 错误处理
- 包体与性能检查

退出标准：

- 可生成可安装 APK
- 完成一轮稳定性验证

## 近期待办

- 实现训练引擎状态机
- 明确页面与文档统一使用“本次训练”
- 设计 Room 表结构
- 设计训练流程页面草图
- 明确动作内容来源策略
- 设计远程构建 APK 的交付流程
